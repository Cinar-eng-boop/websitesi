# Otomobil Web Sitesi

Bu proje HTML, CSS ve JavaScript kullanılarak yapılmıştır.

## Özellikler
- Modern ve responsive tasarım
- Otomobil kataloğu
- İletişim formu
- Smooth scrolling navigasyonu
- Mobil uyumlu yapı

## Dosya Yapısı
```
websitesi/
├── index.html      # Ana sayfa
├── style.css       # Stil dosyası
├── script.js       # JavaScript dosyası
└── README.md       # Bu dosya
```

## Kurulum ve Çalıştırma
1. Dosyaları bilgisayarınıza indirin
2. `index.html` dosyasını çift tıklayarak veya tarayıcıda açarak çalıştırın
3. Alternatif olarak, bir web sunucusunda (Apache, Nginx) barındırabilirsiniz

## Teknolojiler
- **HTML5**: Sayfa yapısı
- **CSS3**: Stil ve responsive tasarım
- **Vanilla JavaScript**: İnteraktif özellikler

## Özellikler Detayları
- **Hero Bölümü**: Büyük başlık ve açıklama
- **Araç Kataloğu**: 3 farklı araç modeli
- **Hakkımızda**: Firma hakkında bilgi
- **İletişim Formu**: Müşteri mesajları için
- **Responsive Navigasyon**: Sayfa içi linkler

## Geliştirme
Bu projeyi geliştirmek için:
1. HTML yapısını `index.html`'de düzenleyin
2. Stilleri `style.css`'de değiştirin
3. JavaScript fonksiyonlarını `script.js`'de ekleyin

## İletişim
Herhangi bir soru için bu depoyu kullanabilirsiniz.
